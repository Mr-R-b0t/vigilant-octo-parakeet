## 🚀 Getting Started

***Dependencies***

Please ensure you have all the dependencies installed on your system.


### 🔧 Installation

1. Clone the reimagined-rotary-phone repository:
```sh
git clone https://github.com/Mr-R-b0t/reimagined-rotary-phone
```

2. Change to the project directory:
```sh
cd reimagined-rotary-phone
```

3. Install the dependencies for the client and server applications:
```sh
► npm i --force
```

### 🤖 Running the client

```sh
► npm run serve
```

### 🤖 Running the server

```sh
► node server.js
```
